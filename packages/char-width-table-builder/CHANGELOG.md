# Changelog

- Measurer: `font` is now required.

### New features

- Export `computeWidthsOfRanges`, `computeKerning`, `selectedRanges`, and
  `shortListOfRanges`,
- Allow injecting stylesheets, useful for measuring web fonts.

- Update Puppeteer dependency.
